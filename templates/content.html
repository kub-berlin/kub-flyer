{{ Beratung für geflüchtete Frauen }}

{{ Die Frauenberatung der KuB richtet sich an **geflüchtete Frauen, die besonderen Schutz brauchen**. }}

{{ Das sind vor allem: }}

  • {{ Schwangere }}
  • {{ Alleinerziehende }}
  • {{ Betroffene von häuslicher Gewalt }}
  • {{ Betroffene von sexualisierter Gewalt }}

{{ Wir beraten: }}

  • {{ bei **Fragen zum Aufenthaltsrecht, zum Asylrecht und zum Sozialrecht** }}
  • {{ bei **psychischen und sozialen Problemen** }}

{{ Wenn Sie das wünschen, können wir Sie auch anonym beraten. Sie müssen uns nicht Ihren Namen sagen. }}

{{ Psychosoziale Beratung }}

{{ Sie können zu uns kommen: }}

  • {{ Wenn Sie schlecht schlafen. }}
  • {{ Wenn Sie sehr oft traurig sind. }}
  • {{ Wenn Sie oft und große Angst haben. }}
  • {{ Wenn Sie sich hilflos fühlen. }}
  • {{ Wenn Sie nicht wissen, wie Sie weiterleben sollen. }}

{{ Wir hören Ihnen zu. Wir suchen mit Ihnen zusammen Wege, wie Sie mit Ihren Problemen umgehen können. }}

{{ Sie entscheiden, worüber wir reden. }}
{{ Sie gehen keine Verpflichtungen ein. }}

{{ Versuchen Sie es einfach! }}
